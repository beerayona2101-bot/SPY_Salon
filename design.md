
# SPY Salon - Design System
Version: 2.0

> A premium luxury design system for SPY Salon built with Next.js, Tailwind CSS, Glassmorphism, Neumorphism, and Framer Motion.

# 1. Brand Vision

SPY Salon should communicate:
- Luxury
- Beauty
- Elegance
- Confidence
- Modern Technology
- Premium Customer Experience

---

# 2. Visual Identity

## Logo Usage
- Always keep clear space equal to the logo icon height.
- Use the gold logo on dark backgrounds.
- Use the black logo on white backgrounds.
- Use the icon-only logo for favicons and mobile navigation.

## Brand Personality
- Premium
- Elegant
- Sophisticated
- Minimal
- Trustworthy

# 17. Do's

- Keep generous whitespace
- Use consistent icon sizes
- Prefer subtle animations
- Maintain premium look

# 18. Don'ts

- Avoid clutter
- Avoid multiple font families
- Avoid harsh shadows
- Avoid inconsistent spacing
- Avoid bright neon colors
